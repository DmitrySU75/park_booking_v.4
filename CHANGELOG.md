# Changelog

## [4.1.0] - 2026-05-06

### Added

- Поддержка разных юридических лиц для оплаты
- API методы: create_order, update_status, update_order, get_orders, get_payment_info, update_prices
- Продление времени аренды администратором
- Скидки и промокоды
- Ограничения по датам (запрет почасового бронирования)
- Уведомления в Telegram
- Мягкое удаление заказов
- Дашборд со статистикой
- История изменения цен

### Changed

- Идентификация беседок по ID вместо названия
- Обновлена логика тарифов (летний/зимний периоды)

## [4.0.0] - 2026-05-04

### Добавлено

- Первоначальный выпуск модуля
- Форма бронирования с выбором даты и типа аренды
- Интеграция с LibreBooking API (проверка доступности, создание бронирований)
- Интеграция с ЮKassa (предоплата, вебхук)
- Интеграция с Битрикс24 (лиды)
- Двусторонняя интеграция с 1С:УНФ (создание, отмена, изменение бронирований)
- Фильтрация каталога по дате
- Ценовые периоды (админка + импорт из 1С)
- Сезонные настройки (летнее/зимнее время)
- Маска ввода телефона
- Административный интерфейс для управления ценовыми периодами

### Технические особенности

- Поддержка PHP 8.2+
- Автоматическое обновление токена API
- Логирование всех операций
- Поддержка изменения домена через конфигурацию

## [1.0.0] - 2026-04-01

### Бета-версия

- Тестирование интеграции с LibreBooking
- Тестирование платежей через ЮKassa
- Отладка двусторонней синхронизации с 1С
