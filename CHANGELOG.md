# Changelog

## [1.0.0] - 2025-01-15

### Добавлено

- Первоначальный выпуск модуля
- Форма бронирования с выбором даты и типа аренды
- Интеграция с LibreBooking API (проверка доступности, создание бронирований)
- Интеграция с ЮKassa (предоплата, вебхук)
- Интеграция с Битрикс24 (лиды)
- Двусторонняя интеграция с 1С:УНФ (создание, отмена, изменение бронирований)
- Фильтрация каталога по дате
- Ценовые периоды (админка + импорт из 1С)
- Сезонные настройки (летнее/зимнее время)
- Маска ввода телефона
- Административный интерфейс для управления ценовыми периодами

### Технические особенности

- Поддержка PHP 8.2+
- Автоматическое обновление токена API
- Логирование всех операций
- Поддержка изменения домена через конфигурацию

## [0.9.0] - 2024-12-01
