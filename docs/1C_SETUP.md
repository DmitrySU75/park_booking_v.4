# Настройка двусторонней интеграции с 1С:УНФ

## 1. Настройка HTTP-сервиса в 1С:УНФ

### 1.1. Создание HTTP-сервиса

В конфигураторе 1С:УНФ:

1. Откройте ветку **«Общие» → «HTTP-сервисы»**
2. Нажмите **«Добавить»**
3. Заполните:
   - **Имя:** `booking`
   - **Шаблон:** `/booking/`
   - **Описание:** «Приём бронирований с сайта»

### 1.2. Создание шаблона URI

Внутри HTTP-сервиса `booking` добавьте шаблон URI:

| Параметр       | Значение          |
| -------------- | ----------------- |
| **Шаблон**     | `/create`         |
| **HTTP-метод** | `POST`            |
| **URL-шаблон** | `/booking/create` |

### 1.3. Код обработчика

```javascript
Процедура ОбработкаЗапросаНаСозданиеБрони(Запрос, Ответ)

    ВходящиеДанные = ПрочитатьJSON(Запрос.ПолучитьТелоКакСтроку());

    // Проверка API-ключа
    Если ВходящиеДанные.api_key <> "секретный_ключ" Тогда
        Ответ.УстановитьКодСостояния(401);
        Возврат;
    КонецЕсли;

    // Создание контрагента
    Контрагент = НайтиИлиСоздатьКонтрагента(ВходящиеДанные.customer);

    // Создание номенклатуры
    Номенклатура = НайтиИлиСоздатьНоменклатуру(ВходящиеДанные.booking.resource_name);

    // Создание документа «Реализация услуг»
    Док = Документы.РеализацияУслуг.СоздатьДокумент();
    Док.Контрагент = Контрагент;
    Док.Дата = ТекущаяДата();

    Строка = Док.Услуги.Добавить();
    Строка.Номенклатура = Номенклатура;
    Строка.Количество = 1;
    Строка.Цена = ВходящиеДанные.booking.total_price;

    Док.Записать();
    Док.Провести();

    // Формирование ответа
    Ответ.УстановитьКодСостояния(200);
    Ответ.УстановитьТелоИзСтроки(ЗаписатьJSON(Новый Структура("success", Истина, "document_number", Док.Номер)));

КонецПроцедуры
```

## 2. Настройка HTTP-клиента

### 2.1. Код для отправки бронирования

```javascript
Процедура ОтправитьБроньНаСайт()

    Данные = Новый Структура;
    Данные.Вставить("api_key", "секретный_ключ");
    Данные.Вставить("action", "create_booking");
    Данные.Вставить("resource_id", Объект.Беседка.Код);
    Данные.Вставить("start_time", Формат(Объект.ДатаНачала, "ДФ=YYYY-MM-DDThh:mm:ss+05:00"));
    Данные.Вставить("end_time", Формат(Объект.ДатаОкончания, "ДФ=YYYY-MM-DDThh:mm:ss+05:00"));
    Данные.Вставить("first_name", Объект.Клиент.Имя);
    Данные.Вставить("last_name", Объект.Клиент.Фамилия);
    Данные.Вставить("phone", Объект.Клиент.Телефон);
    Данные.Вставить("email", Объект.Клиент.Email);
    Данные.Вставить("deposit_amount", Объект.СуммаПредоплаты);

    HTTPСоединение = Новый HTTPСоединение("ваш-домен.ru", 443);
    HTTPЗапрос = Новый HTTPЗапрос("/local/modules/avs_booking/api_1c.php");
    HTTPЗапрос.Метод = "POST";
    HTTPЗапрос.Заголовки.Вставить("Content-Type", "application/json");
    HTTPЗапрос.Заголовки.Вставить("X-API-Key", "секретный_ключ");
    HTTPЗапрос.УстановитьТелоИзСтроки(ЗаписатьJSON(Данные));

    HTTPОтвет = HTTPСоединение.ОтправитьДляОбработки(HTTPЗапрос);

КонецПроцедуры
```

## 5. Устранение неполадок

| Параметр                    | Значение                                            |
| --------------------------- | --------------------------------------------------- |
| **401 Unauthorized**        | Проверьте API-ключ в настройках модуля и 1С`        |
| **404 Not Found**           | Проверьте URL HTTP-сервиса в 1С                     |
| **409 Conflict**            | Время уже занято, проверьте расписание LibreBooking |
| **Не создаётся контрагент** | Проверьте заполнение телефона и email               |
